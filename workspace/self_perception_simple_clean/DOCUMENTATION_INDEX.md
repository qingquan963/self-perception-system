## 🔍 搜索技巧

### 按关键词搜索
在文档中搜索特定关键词：

```bash
# 在文档中搜索"安装"
grep -r "安装" *.md

# 搜索"API"（不区分大小写）
grep -ri "api" *.md

# 搜索"错误"并显示上下文
grep -B2 -A2 "错误" *.md
```

### 按章节搜索
每个文档都有清晰的章节结构，使用以下格式快速定位：

```
# 主标题
## 二级标题
### 三级标题
```

### 使用文档链接
所有文档都支持Markdown内部链接，可以直接跳转到特定章节。
